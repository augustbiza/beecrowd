## Explicação C
```c
int a, b;
```
Declaração de dois dados do tipo **int**
```c
scanf("%d%d", &a, &b);
```
Atribuição de valores **int** aos dois dados através da classe Scanner
```c
printf("X = %d\n", a+b);
```
Imprime a soma

## Explicação C++
```cpp
int a, b;
```
Declaração de dois dados do tipo **int**
```cpp
std::cin>>a>>b;
```
Atribuição de valores **int** aos dois dados através da classe Scanner
```c
std::cout<<"X = "<<a+b<<std::endl;
```
Imprime a soma
